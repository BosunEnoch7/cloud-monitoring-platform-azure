# Screenshots

This directory is reserved for curated visual evidence of the working platform.

Recommended screenshots:

- Azure resource group and VM overview
- GitHub Actions successful Terraform apply
- GitHub Actions successful observability validation
- Prometheus targets page
- Prometheus alert rules page
- Alertmanager status or alert list
- Grafana `Cloud Monitoring - Node Overview` dashboard
- Email notification evidence after SMTP is activated

Screenshots must be reviewed for subscription IDs, public IP addresses, email addresses, tokens, and other sensitive information before they are committed.
